# Base de conocimiento — FocusWeb Chile

Este set de archivos markdown le da contexto de negocio a un agente de IA (asistente web / chatbot) sobre **FocusWeb**, la empresa de SEO técnico, optimización de velocidad y desarrollo web de Felipe Ibar, con base en Praga y clientes en todo Chile.

**Fuente:** contenido extraído de focusweb.cl (julio 2026).
**Importante:** los precios son "desde" y deben tratarse como referencia, no como cotización cerrada. Ver `04-precios-resumen.md` y las reglas de comportamiento en `AGENTS.md`.

## Cómo mantenerlo actualizado

Este contenido se extrajo directamente del sitio en julio 2026. Si cambian precios, servicios, textos o el número de WhatsApp en focusweb.cl, estos archivos hay que actualizarlos a mano — no se sincronizan solos con el sitio en vivo.

## Alcance cubierto vs. no cubierto

**Cubierto:** Home, los 3 servicios principales, landing pages standalone, nuestro proceso, metodología/diagnóstico inicial, mi historia (Felipe), contacto.

**No cubierto todavía (si el agente necesita responder sobre esto, avisar que falta cargarlo):** contenido específico de los posts del blog (solo están linkeados como recursos en `06-preguntas-frecuentes.md`), `/crear-pagina-web-chile`, `/paginas-web-chile` y `/check-list-ecommerce` como páginas completas.
