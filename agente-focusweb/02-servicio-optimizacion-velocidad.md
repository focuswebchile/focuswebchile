# Servicio: Optimización de Velocidad Web

## Qué es

Trabajo técnico que mejora cómo carga, responde y se comporta el sitio en condiciones reales: recursos, carga crítica, estabilidad visual, caché y señales de SEO técnico. No es solo "subir una nota en PageSpeed".

## Qué problemas resuelve

- Rebote alto en móvil, poco tiempo de permanencia.
- Core Web Vitals en rojo (afectan experiencia y confianza).
- Pérdida de tracción SEO frente a competidores más rápidos.
- Sitio que convierte poco aunque tenga buen tráfico.

Dato del sitio: **53% de usuarios móviles abandona sitios que tardan más de 3 segundos**, y en Chile el 70% del tráfico web es móvil (fuente: Google Mobile Study). Probabilidad de rebote adicional: +32% a los 3s, +90% a los 5s, +123% a los 10s vs. 1s base.

## Qué incluye

1. **Diagnóstico de performance real:** PageSpeed, Lighthouse, métricas de usuario.
2. **Optimización de recursos críticos:** diferir scripts, CSS crítico, limpieza de recursos que frenan la carga.
3. **Imágenes y medios:** compresión, formatos modernos (WebP), lazy loading.
4. **Caché y entrega:** headers, expiraciones, CDN según el stack.
5. **Core Web Vitals + SEO técnico:** LCP, interacción, estabilidad visual.
6. **Reporte antes/después:** evidencia de mejora y próximos pasos.

## Caso de referencia (resultados reales de un proyecto)

| Métrica | Antes | Después |
|---|---|---|
| PageSpeed móvil | 48 | 88 |
| LCP | 4.9s | 2.4s |

| TBT | 470 ms | 110 ms |
| Speed Index | 3.8s | 1.4s |

## Cuándo conviene optimizar vs. rehacer

**Conviene optimizar si:** la web ya cumple su función comercial pero carga lento; el problema es performance/Core Web Vitals/configuración técnica; se quiere mejorar sin partir de cero.

**Conviene rehacer si:** la plataforma actual limita cambios importantes; la arquitectura está desordenada desde la base; se necesita algo más amplio que velocidad técnica.

Si no está claro, el primer paso recomendado es la auditoría (ver `01-servicio-auditoria-seo.md`).

## Qué NO incluye

- Rediseño completo del sitio.
- Estrategia editorial profunda de contenidos.
- Gestión de backlinks y outreach.

## Precio y tiempos

**Optimización estándar** (ideal para WordPress y sitios pymes): **Desde $110.000 CLP**, tiempo de 1-2 semanas.

Diagnóstico inicial: 48h. Implementación: 1-2 semanas.

## FAQs clave

- **¿Garantizan posiciones en Google?** No. Se garantiza mejora técnica medible; el posicionamiento final también depende de contenido, competencia y autoridad de dominio.
- **¿Qué necesito para empezar?** URL del sitio y acceso técnico básico (CMS/hosting o soporte del proveedor). Si no lo tiene, se lo guía paso a paso.
