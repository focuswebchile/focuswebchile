# Voice Guide — Focus Web

> Este documento define la personalidad de la marca: quién es Felipe, cómo
> piensa, cómo suena y cómo no suena. Es la base de la que dependen
> `Homepage.md` (cómo se aplicó esta voz a la home) y `COPY.md` (reglas
> operativas para generar cualquier contenido nuevo). Si un texto nuevo
> tiene que sonar a Focus Web, este es el documento que hay que leer
> primero — no las páginas ya escritas.

## 1. Quién es Felipe

Felipe es la persona detrás de Focus Web — no una agencia, no un equipo,
no una consultora. Ayuda a pymes chilenas a tener presencia web que
realmente funcione. La marca no existe separada de la persona: cuando
alguien lee el sitio, habla con Felipe, no con "el equipo de Focus Web".

## 2. Qué es Focus Web

**No construye páginas web. Se hace responsable de que funcionen.**

Esta es la idea que sostiene todo lo demás — el porqué detrás de cada
decisión de la marca:

- No vende paquetes → porque primero necesita entender el problema real.
- No entrega una web y desaparece → porque sigue acompañando después.
- No habla primero de SEO → porque primero le importa el negocio, no la
  herramienta.
- No muestra funcionalidades → porque primero quiere resolver algo
  concreto.

El trabajo real no es "diseño web + SEO + automatización" — eso es lo que
hace, no lo que es. Lo que es: **reducir la incertidumbre del cliente**
sobre su presencia digital.

## 3. Qué NO es Focus Web

- No es una agencia de marketing digital genérica.
- No es un vendedor de servicios ni de paquetes cerrados.
- No es un copywriter tratando de sonar persuasivo.
- No es un consultor hablando en términos de "implementación",
  "optimización" o "validación".
- No pretende impresionar — si una frase suena a que quiere impresionar,
  probablemente no es de Focus Web.

## 4. Cómo piensa Felipe (esto es lo que faltaba)

No empieza preguntándose qué página necesita un cliente. Empieza
preguntándose **qué problema está intentando resolver.**

Una página web nunca es el objetivo — es una herramienta entre varias
posibles:

- Si una landing resuelve mejor el problema que un sitio completo,
  recomienda una landing.
- Si una automatización genera más valor que una sección nueva en la web,
  prefiere automatizar.
- Si un cliente todavía no necesita SEO, no le vende SEO.

**No vende más servicios — intenta tomar buenas decisiones con el
cliente.** Al escribir para Focus Web, nunca pensar primero en diseño,
desarrollo, SEO o automatización — pensar primero en el negocio del
cliente, y después decidir qué herramienta tiene sentido para ese caso.

## 5. Cómo debe sentirse leer algo de Focus Web

Después de leer cualquier texto de la marca, alguien debería pensar:

- "Este tipo sabe de lo que habla."
- "Nunca intentó impresionarme."
- "Me explicó algo complejo de forma simple."
- "No sentí que quisiera venderme algo."
- "Probablemente me diría si algo no vale la pena."

Si un texto no genera esa sensación, todavía no representa la voz de la
marca — sin importar cuántas reglas de estilo haya cumplido.

## 6. Principios editoriales

1. **Habla de resultados antes que de servicios.** El lector no necesita
   saber qué hace Felipe primero — necesita saber qué cambia para su
   negocio.
2. **Explica la forma de trabajar antes que la lista de capacidades.**
   "Cómo trabajo" genera más confianza que "qué sé hacer".
3. **Verbos cotidianos, no de consultoría.** Preferir "entender", "probar",
   "acompañar" — evitar "implementar", "validar", "optimizar".
4. **Entre sonar técnico y sonar cercano, gana siempre lo cercano.**
5. **Una idea por párrafo.** Meter varias ideas en una misma oración hace
   que el lector no recuerde ninguna. "No te entrego una web y
   desaparezco." Punto. Eso se recuerda. Una oración con cinco conceptos
   encadenados, no.
6. **Segunda persona, siempre que se pueda** — le habla directamente al
   lector, no describe a un cliente abstracto en tercera persona.

## 7. Test rápido antes de escribir o revisar una sección

Antes de dar por bueno un párrafo, preguntarse:

**¿Estoy hablando del servicio, o estoy hablando del problema del
cliente?**

Si la respuesta es "servicio", el texto probablemente todavía puede
mejorar. Si habla del problema y de cómo cambia la situación del cliente,
va por buen camino.

Prueba complementaria: **¿esta frase podría haberla dicho Felipe en una
videollamada real con un cliente?** Si suena a algo que se diría en una
llamada, es la voz correcta. Si suena a algo escrito para una landing
page genérica, no lo es.

## 8. Ejemplos de frases que representan la marca

- "No hago solamente páginas web."
- "No vendo paquetes. Diagnóstico primero."
- "Y aquí no desaparezco."
- "Antes de darte un presupuesto, necesito entender el problema."
- "Si algo no aporta valor, prefiero decirte que no lo hagas."

## 9. Frases que nunca deberían aparecer

- Cualquier variante de "diagnóstico estratégico", "implementación
  técnica", "publicación y validación", "optimización continua" — suena
  a proceso de consultora, no a Felipe explicando cómo trabaja.
- Enumeraciones de servicios inmediatamente después de decir "no hago
  solamente X" — contradice la idea que se acaba de plantear.
- "Precios transparentes", "Todo Chile", o cualquier bullet que podría
  decir cualquier competidor sin cambiar una palabra.
- Cualquier frase que describa el proceso de trabajo en sustantivos
  abstractos en vez de verbos concretos de lo que Felipe realmente hace.
